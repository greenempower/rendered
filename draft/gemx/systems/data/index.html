GEMX Systems: Data

GEMX will store all data as objects and each object is a file stored in /data/objects/

The file objects are named semi-randomly.

There will be a registry and a general system for keeping meta data and dealing with objects in an intelligeble manner.

Each object will have tags and identifiers associated with it.

All objects are compressed and encrypted.

Objects will be compressed via an intelligent and semi-autonomous (ML like) algorithm leveraging hard-coded intel and aggregate data known to the system.
